# SIRA Banking App

This project now runs without the Base44 SDK.

## Local runtime

- Data is stored in the browser with a local in-app client.
- Demo users, applications, contacts, audit logs, uploads, CSV export, and PDF generation all run locally.
- No Base44 app configuration or Base44 Vite plugin is required.

## Commands

```bash
npm install
npm run dev
```

## Notes

- Clearing browser storage resets the local demo data.
- The app boots with a seeded demo session so the dashboard, admin, and internal workflows can be explored immediately.
